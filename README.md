# Forecast Verification

Compares IFS and AIFS 2m temperature forecasts against ERA5 reanalysis over Europe.

Computes MAE, RMSE, and R^2 at each forecast lead time (0-360h).

## Setup

Install dependencies:

```
pip install -r requirements.txt
```

## Running

Open and run `main.ipynb` in Jupyter:

```
jupyter notebook main.ipynb
```

Data files are already included in `data/`.
